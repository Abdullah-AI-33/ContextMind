# ContextMind

Building a document intelligence system from scratch — starting with the math foundations, ending with a production-grade agentic RAG system.

This repository is my public build log as I follow a 24-month self-study plan to become an AI/ML engineer. I'm an AI and Robotics Engineering student at Al-Nahrain University, and this project runs alongside my coursework as my primary hands-on learning vehicle.

I'm documenting every phase here and on LinkedIn as I go — this is a live project, not a finished product. Expect the code and structure to evolve significantly over time.

## Why this project

I wanted to learn AI engineering by building something real end-to-end, rather than only completing isolated course exercises. ContextMind is designed to grow in complexity as my skills grow: starting with basic classification, and evolving toward a full agentic RAG (Retrieval-Augmented Generation) system with multi-agent architecture and multimodal understanding.

## Roadmap
